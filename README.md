-   `outDir`: 컴파일 결과물을 저장 할 위치
-   `esModuleInterop`: 모듈 시스템을 _commonjs_ 를 사용하고 있다면 `true`로 설정
-   `resolveJsonModule`: _JSON_ 을 사용 할 경우 `true`로 설정
